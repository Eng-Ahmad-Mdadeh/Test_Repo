# NeuroBalance Flutter App

تطبيق **NeuroBalance** هو منصة تقييم نفسي مبنية بإطار Flutter وتعتمد على مبدأ **Clean Architecture** من أجل الفصل بين طبقات المجال (Domain) والبيانات (Data) والعرض (Presentation). يوفر التطبيق تجربة استخدام عربية (RTL) ولوحة معلومات حديثة مع الوضع الليلي، إلى جانب صفحات للمرضى وجلسات التقييم والرؤى التحليلية.

## المتطلبات المسبقة
- تثبيت [Flutter](https://docs.flutter.dev/get-started/install) (الإصدار 3.13 أو أحدث).
- تمكين دعم منصات الويب أو الأجهزة المستهدفة عبر `flutter config` بحسب حاجتك.

## تشغيل المشروع
```bash
flutter pub get
flutter run -d chrome   # لتشغيل نسخة الويب
```

## هيكلية المجلدات
```
lib/
  domain/
    entities/          # الكيانات الأساسية للمجال
    repositories/      # تعريف الواجهات للمستودعات
    usecases/          # حالات الاستخدام (منطق المجال)
  data/
    repositories/      # تطبيقات المستودعات (Mock)
  presentation/
    state/             # المراقبون (ChangeNotifier) وإدارة الحالة
    pages/             # الشاشات الرئيسية للتطبيق
    widgets/           # المكونات المشتركة بين الشاشات
    theme/             # التخصيص اللوني والخطوط
```

## الميزات الرئيسية
- لوحة تحكم تعرض مؤشرات يومية وأسبوعية للتقدم العلاجي.
- قائمة مرضى مع حالة التقييم وخطط المتابعة.
- تدفق تقييم تفاعلي مع أسئلة متدرجة ومستويات حدّة.
- صفحة رؤى تحليلية تتضمن رسوم بيانية مبسّطة ونقاط اتخاذ قرار.
- دعم كامل لاتجاه الكتابة من اليمين إلى اليسار.

## الاختبارات
لم يتم تضمين اختبارات تلقائية في الوقت الحالي. بإمكانك إضافة اختبارات وحدات لكل حالة استخدام ضمن مجلد `test/` لاحقًا.
