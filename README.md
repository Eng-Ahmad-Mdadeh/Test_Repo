# NeuroBalance – منصة التقييم النفسي الذكي

تطبيق ويب تفاعلي يعتمد على مبادئ **Clean Architecture** لتقديم تجربة تقييم نفسي رقمية تدعم اللغة العربية، العمل دون اتصال، والتكامل مع التحليلات المعتمدة على الذكاء الاصطناعي. تم تصميم الواجهات استنادًا إلى تحليل المشروع المرفوع بالصور، مع التركيز على رحلة المريض، أتمتة التقارير، وحماية البيانات.

## نظرة معمارية

يعتمد المشروع على بنية متعددة الطبقات تفصل بين المجال المنطقي وطبقات العرض والبيانات:

```
src/
├── domain/           # الكيانات وتعريفات المستودعات (Entities & Repositories)
├── application/      # حالات الاستخدام (Use Cases)
├── infrastructure/   # مصادر البيانات والتنفيذات الواقعية للمستودعات
└── presentation/     # الواجهات والتنسيقات ومسارات التصفح
```

- **Domain**: يعرف الكيانات مثل المريض، المقاييس، ونتائج التقييم.
- **Application**: يحتوي على حالات الاستخدام مثل جلب المقاييس، إرسال التقييم، وإحصاءات لوحة التحكم.
- **Infrastructure**: تنفيذ تجريبي (Mock) لمصدر البيانات يحاكي تدفق العمل دون اتصال.
- **Presentation**: تطبيق React بواجهات تفاعلية عصرية، دعم RTL، وتدرجات لونية مستوحاة من بيئة العيادات الذكية.

## الميزات الرئيسية

- لوحة تحكم فورية بالمؤشرات الرئيسية وعدد التنبيهات الحرجة.
- تجربة تقييم تفاعلية تدعم اللمس، وضع دون اتصال، ومساعد ذكاء اصطناعي.
- ملفات مرضى غنية بالمعلومات مع تصفية حسب مستوى الخطر.
- مختبر تحليلات يقدم رسومًا بيانية، خريطة رحلة المريض، وتلخيص جاهزية العمل دون اتصال.

## البدء محليًا

> يتطلب Node.js 18 أو أحدث.

```bash
npm install
npm run dev
```

سيتم تشغيل التطبيق عبر Vite على العنوان `http://localhost:5173`.

لإنشاء نسخة إنتاجية:

```bash
npm run build
```

## التصميم وتجربة المستخدم

- تم اعتماد خطوط Manrope مع نظام ألوان ليلي (Dark Mode) يبرز درجات الأزرق البنفسجي لتعزيز الإحساس بالثقة.
- بطاقات بمساحات بيضاء كبيرة وتباين عالٍ لسهولة القراءة وتقليل إجهاد العين خلال الجلسات الطويلة.
- عناصر تفاعلية بظلال ناعمة، تدرجات ديناميكية، ومؤشرات لحظية لحالة الاتصال.
- تدعم الواجهات RTL بشكل افتراضي مع نصوص عربية معاصرة، وتصنيف واضح للحالات (حرج، متابعة، مستقر).

## الخطوات القادمة

- ربط المستودع بمصدر بيانات حقيقي أو واجهة برمجة تطبيقات سحابية.
- إضافة طبقة مصادقة متعددة العوامل وفق سياسات الأمان.
- دعم إحصاءات زمنية أوسع وتكامل مع أجهزة استشعار فعلية (HRV، كاميرات حرارية).
